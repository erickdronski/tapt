# 21 — EXECUTION PLAN (synthesized from 14 audits, 2026-07-12)

One merged, deduped, prioritized plan from the full audit round: data (beers,
imagery, venues), market engine verification, UX (Home, Market, Cellar,
Discover, You), games, partner backbone, web offerings, legal, content.
Every item is verified in code or live prod SQL by the source audit; see the
"where" column for the anchor. Effort: S = under half a day, M = half day to
two days, L = multi-day.

Tags: [HONESTY] = the current behavior contradicts hard rule 1 (nothing
invented / never fabricate movement). [VOICE] = contradicts hard rule 2
(plain voice, no number-flexing, no hype). These are called out explicitly
because they are owner directives, not style preferences.

Prod-write note: anything touching `supabase/migrations/` ships as a mirrored
migration applied via MCP in the same commit, per AGENTS.md. Items needing an
owner env var, account, money, or attorney are in OWNER-GATED even if the code
can be written now.
